Privacy Policy

Game: Ragdoll Drop • Last updated: September 10, 2025

1) Overview

This Privacy Policy explains how the game Ragdoll Drop (“the game”, “we”, “our”) handles information. The game is designed to respect player privacy and does not collect personal data.

2) Data Collection

  • No personal data: The game does not collect, store, or share personal information about players.
  • No account required: The game does not require registration, login, or any form of identification.
  • No cookies/analytics: The game does not use cookies, analytics, advertising SDKs, or tracking technologies.

3) External Requests (api.ericetto.com)

To enable a specific gameplay feature (Level 14), the game makes read-only requests to the domain ericetto.com to retrieve Bitcoin price history.

  • These requests fetch public market data only and are used exclusively to generate in-game content (a bar-chart mechanic that launches the character).
  • No personal data is included in these requests.

4) Server Logs

When contacting ericetto.com, standard web server logs may record technical information (e.g., IP address, user agent, timestamp, requested URL) for security, anti-abuse, and operational purposes. This information is not used to identify players and is not combined with any other data to profile individuals.

5) Third Parties & Sharing

The game does not share player data with third parties. The external resource ericetto.com provides numerical market data only.

6) Children’s Privacy

The game is suitable for players of all ages as it does not collect personal data.

7) Changes to This Policy

If we make material changes to this Privacy Policy, we will update the “Last updated” date above. Continued use of the game after changes take effect constitutes acceptance of the revised policy.
